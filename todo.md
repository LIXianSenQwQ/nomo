# 考虑实现

**问题 7：段落菜单里的列表区怎么取舍？**

我的推荐：**保留有序列表、无序列表、任务列表；暂时不放任务状态和列表缩进**。\
这样更接近 Typedown 的轻量结构。任务状态 >、列表缩进 > 属于进阶操作，一级菜单会变长，而且很多时候可以通过快捷键或编辑器行为处理。

你选：

1. **只保留有序列表、无序列表、任务列表**
2. **加上任务状态 >**
3. **加上列表缩进 >**
4. **任务状态和列表缩进都加上**
